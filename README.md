# AI Activation Copilot

A simple Streamlit prototype for generating a 90-day Snowflake activation blueprint from customer discovery inputs.

## Run locally

```bash
cd ai_activation_copilot
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

## Windows

```bash
cd ai_activation_copilot
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

## What it generates

- Activation summary
- Recommended architecture
- Thin-slice MVPs
- 30-60-90 roadmap
- Day 90 handover recommendation
- Suggested agent templates
